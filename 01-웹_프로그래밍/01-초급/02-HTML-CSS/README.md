## HTML

- HyperText Markup Language
  - Hypertext : 정보가 담긴 텍스트
  - Markup : tag
    - h1, h2, h3, h4, h5, h6
      - heading (머릿말, 제목)
    - p
      - Paragraph (문단)
    - ul
      - `U`nordered `L`ist (순서가 없는 리스트)
    - li
      - List
    - 사용방법
      - `<h1>제목</h1>`
      - `<p>문단</p>`
  - `<html>`
    - `<head>` : meta 정보 (문서의 핵심 정보)
    - `<body>` : 문서의 내용
